# Git Basics

Git started with Git
```
git init
```

Whats goin on?
```
git status
```

Stage 1 file or all files
```
git add readme.md
git add -A
```

Commit
```
git commit -m "Initial commit"
```

Add a connection to github
```
git remote add origin https://github.com/Aobatrouz/CmdLineDemo.git
```

Push to github
```
git push origin master
```
